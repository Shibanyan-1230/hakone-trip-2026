SKULL（スカル）
🕑 15〜30分 / 🎲 ブラフ系
BASIC RULES（4人プレイ）
  1. 各プレイヤーに花3枚・ドクロ1枚の計4枚を配る。自分だけ中身を確認OK
  2. 裏向きに1枚ずつ自分の前に重ねて置く（花でもドクロでもOK）
  3. 自分の番で「もう1枚置く」か「チャレンジ宣言（数字を宣言）」を選ぶ
  4. チャレンジ宣言：「自分は○枚めくれる！」と宣言。他のプレイヤーはより大きい数で上乗せするか、パスする
  5. 最も大きい数を宣言した人が、自分の山から先にその枚数をめくる。全て花なら成功！
  6. ドクロをめくったら失敗 → ランダムに手札1枚を失う
🏆 勝利条件：2回チャレンジ成功で勝ち！

  4. インサイダーは答えを知っているので、バレないように上手く誘導する
  5. お題が当たったら → 投票タイム！「誰がインサイダーか？」を議論して多数決

UNO FLIP!

BASIC RULES（4人プレイ）
  1. 基本はUNOと同じ！色か数字が合うカードを出す。出せなければ1枚引く
  2. カードにライトサイド（明るい色）とダークサイド（暗い色）の2面がある
